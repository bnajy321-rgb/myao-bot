# 🪪 نظام ابشر المياااااو

بوت Discord لنظام ابشر المياااااو

## التثبيت

```bash
npm install
```

## الإعداد

1. انسخ `.env.example` إلى `.env`
2. أضف التوكن الجديد في `.env`:

```env
DISCORD_TOKEN=your_new_token_here
CLIENT_ID=1514128685970755635
GUILD_ID=1512533943059091678
```

## التشغيل

```bash
node bot.js
```

- `/هوية <رقم>` - عرض معلومات الهوية
